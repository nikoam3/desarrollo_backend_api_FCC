Timestamp Microservice

Descripción

Esta API convierte fechas entre formato Unix (milisegundos) y formato UTC. Soporta fechas en formato ISO (ej. "2015-12-25") y timestamps Unix (ej. "1451001600000"). Si no se proporciona una fecha, devuelve la fecha actual.

Ejemplos de uso:

  • https://desarrollobackendapifcc-production.up.railway.app/api/2015-12-25
  • https://desarrollobackendapifcc-production.up.railway.app/api/1451001600000
  • /api/ Fecha actual

Ejemplo de salida:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Para fechas inválidas:

{"error":"Invalid Date"}
